After Flashing

01
Connect to PlaneRadar Wi-Fi
The device broadcasts a setup network. Join PlaneRadar on your phone and a setup page opens automatically.
02
Enter your Wi-Fi credentials
Type your home network name and password. Optionally enter a ZIP code — or leave blank to auto-locate via IP.
03
Flights appear in ~30 seconds
The device locates you, pulls live ADS-B data, and shows every aircraft within your search radius sorted by distance.
04
Use the tab bar to navigate
ALL shows all traffic. AIRLINES filters to commercial flights. SETTINGS lets you change location, radius, and update interval.

What You Get

✈
Live ADS-B Data
Pulls from airplanes.live, with automatic fallback to adsb.fi and OpenSky Network. No API key needed.
📍
Auto Location
Detects your location via IP on first boot, or enter a ZIP code for precision.
+=-
Altitude Trend
Green for climbing, red for descending, grey for level flight — visible at a glance on every row.
tap
Detail Card
Tap any flight row for a full detail card: registration, altitude, speed, heading, and vertical speed.
📡
No Cloud, No Account
Uses free community ADS-B aggregators. No sign-up, no API key, no subscription required.
