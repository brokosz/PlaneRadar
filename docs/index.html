After Flashing

01
Connect to PlaneRadar Wi-Fi
The device broadcasts a setup network. Join PlaneRadar on your phone and a setup page opens automatically.
02
Enter your Wi-Fi credentials
Type your home network name and password. Optionally enter a ZIP code — or leave blank to auto-locate via IP.
03
Flights appear in ~30 seconds
The device locates you, fetches from FlightRadar24, and displays all aircraft within your search radius, sorted by distance.
04
Tap ⚙ to adjust settings
Change location, ZIP code, search radius (25–200 mi), and update interval anytime from the on-device settings screen.

What You Get

✈
Live Flight Data
FlightRadar24 data refreshed every 30 seconds. Falls back to OpenSky Network automatically.
📍
Auto Location
Detects your location via IP on first boot, or enter a ZIP code for precision.
↑↓
Altitude Trend
Green arrow for climbing, red for descending, grey for level flight — at a glance.
⚙
On-Device Settings
Touch-based settings screen. Change ZIP code, radius, and refresh rate without reflashing.
📡
No Cloud, No Account
Uses free public APIs. No sign-up, no API key, no subscription required.
🖥
Same Hardware as Halo-F1
Built for the JC4827W543 board — ESP32-S3 with 4.3" capacitive touch TFT.
